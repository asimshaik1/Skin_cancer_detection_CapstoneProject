# 🧠 Skin Cancer Detection using Deep Learning

This project is a deep learning-based web application for detecting skin cancer using dermoscopic images.

---

## 🚀 Features

- Binary & Multi-class classification
- CNN models: VGG16, ResNet50, DenseNet121, Xception
- Real-time prediction using Flask
- Image upload interface
- Grad-CAM visualization for model explainability

---

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- Flask
- HTML, CSS, JavaScript

---

## 📊 Datasets Used

- HAM10000
- BCN20000
